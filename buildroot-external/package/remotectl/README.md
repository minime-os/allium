# remotectl

Remote MinUI input and capture helper for RG35xxSP.

Installed files:
- `/usr/bin/remotectl`
- `/usr/bin/remote-inputd`
- `/usr/bin/remote-fbgrab`
- `/usr/bin/remote-ppm2png`
- `/usr/bin/remote-screenrecctl`
- `/usr/bin/remote-selfcheck`
- `/etc/init.d/S15remotectl`
- `/etc/default/remotectl`

Defaults:
- Socket: `/tmp/remotectl.sock`
- Artifacts: `/mnt/sdcard/system/logs/remotectl`
